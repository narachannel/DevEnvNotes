# Shortcuts

## Visual Studio

- Git Commit : `Ctrl` + `0` + `G`
- Add `;` to end of the line : `Shift` + `Enter` (`Ctrl` + `Shift` + `Enter` for no new line)
- Back / Forward pages during reference jump  `Ctrl` + `-` (+ `Shift`)

## VS Code

- Choose line : `Ctrl` + `l`
- Choose all searching words : `Enter` + `Alt`

## Windows Terminal

- Command palette : `Ctrl` + `Shift` + `p`
- Clear line : `Ctrl` + `u`
- Copy : `Ctrl` + `y`
